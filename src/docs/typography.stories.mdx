# Typography

<br /><br />
#### All typography variants are defined in the theme, and can be passed as props to any Box component by their key name.

```jsx
import { Box } from '../core/components/Box/Box';

// Basic usage
<Box typographyVariant="body1" />
<Box typographyVariant="body3" />

// Responsive usage
<Box typographyVariant={{ _: 'body1', md: 'body5' }} />
<Box typographyVariant={['body1', null, 'body5']}/>
```

#### You should always aim to use a known Typography Variant. However, you have the ability to control individual text style properties
```jsx
import { Box } from '../core/components/Box/Box';

// Use the `body1` typography variant as a base, then override the line height
// `theme.lineHeights['short']` => 1.3
<Box typographyVariant="body3" lineHeight="short" />

// You can also provide a non-named token as a value to `lineHeight`
<Box typographyVariant="body3" lineHeight={1.3} />
```
