# Spacing

<br /><br />
#### We use Styled System, which comes powers-of-two based [spacing scale](https://styled-system.com/guides/why-powers-of-two): 


```jsx
import { Box } from '../core/components/Box/Box';

// set margin-left + margin-right to 4 => 2rem (32px) 
<Box marginX={4}/>

// set padding to 4 => 2rem (32px)
<Box padding={4} />
```
